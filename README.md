
# Autonomous Sanitization Robot

## 📌 Project Overview
The Autonomous Sanitization Robot is an ESP32-based intelligent robotic system designed to perform automatic sanitization in indoor environments. The robot can navigate autonomously, detect obstacles, identify human presence for safety, and perform sanitization operations with minimal human intervention.

---

## 🎯 Objectives
- Reduce human exposure to contaminated environments.
- Automate sanitization processes.
- Improve safety through human detection.
- Enable remote monitoring using IoT technology.

---

## 🚀 Features
- Autonomous Navigation
- Obstacle Avoidance using Ultrasonic Sensor
- Human Detection using PIR Sensor
- Automatic Sanitization System
- IoT Monitoring using Blynk
- ESP32-Based Control
- Real-Time Operation

## ⚙ Working Principle
1. The robot moves autonomously in the environment.
2. Ultrasonic sensor detects obstacles and changes direction.
3. PIR sensor detects human presence.
4. Sanitization process stops when a human is detected.
5. Relay module controls the sanitizer pump.
6. Blynk application provides monitoring and control.
